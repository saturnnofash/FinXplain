## ✅ **Step 5: Generate Synthetic Dataset**

### ✔ What to do:

Create Filipino-like financial data.

### ✔ How:

```python
import numpy as np

n = 1000

data = pd.DataFrame({
    "age": np.random.normal(29, 5, n),
    "income": np.random.lognormal(mean=10, sigma=0.5, size=n),
    "balance": np.random.gamma(2, 2000, n),
    "spending_ratio": np.random.beta(2, 5, n)
})
```

### ✔ Add rules:

```python
data["product"] = np.where(data["balance"] > 10000, "investment", "savings")
```
